# Portfolio Website (local dev)

This is a minimal Next.js + Tailwind scaffold that renders markdown from your workspace.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- The site reads markdown from these folders (relative to project root):
  - `GASTeamStuff/GasTeamStuff`
  - `OarWeThereYetStuff/write-ups`
  - `RoboticsStuff/roboticsObsidian`
- Your photo at the repo root `PhotoOfMe.jpg` is served by `/api/photo` and displayed on the About page.
