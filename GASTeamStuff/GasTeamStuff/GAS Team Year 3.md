This is the current year for me (Fall 2025-Spring 2026). I've spent most of the time researching **Data Products**, which is a feature of FPrime where you can save values to the satellite in non-volatile memory, and downlink it to the ground station computer. There's not much documentation on the feature as it's very new, so I had to communicate with the FPrime team via their GitHub discussions page, and I figured it out and got a prototype working with **Data Products**. Now they'll be able to use this knowledge to use data products in the future.
